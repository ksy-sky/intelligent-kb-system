# Для работы ИИ-ассистента требуется Ollama — локальный сервер языковых моделей.

## Установка Ollama
### Шаг 1 - установка 
1. Скачайте установщик с официального сайта: https://ollama.com/
2. Запустите установщик и следуйте инструкциям
3. После установки Ollama запустится автоматически 
### Шаг 2 - cкачивание модели 
1. Откройте командную строку и выполните:
```
ollama pull llama3.2
```
*Модель занимает ~2 ГБ. Время скачивания зависит от скорости интернета (обычно 5-15 минут).*

2. Проверка установки
```
ollama list
```
*Должна отобразиться модель llama3.2:latest.*
 - Альтернативные модели
Если llama3.2 не подходит, можно использовать другие:
#### Компактная модель (1.6 ГБ)
```
ollama pull phi
```
#### Русскоязычная модель
```
ollama pull saiga
```
*После скачивания измените MODEL_NAME в файле src/scenario_assistant.py на нужную модель.*

#### Работа без Ollama
Если Ollama не установлен, ассистент автоматически переключится в режим локального поиска по базе знаний, а функциональность Сценариев 1 и 2 доступна всегда.
